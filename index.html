All About Node.js

Posted by: Kaitlyn Davis

What is node? Node is an open source Javascript framework. It is used to develop network applications. Node uses a model based on an event loop. It is non-blocking this means it does not make the program wait, it just registers a callback and the program continues.


Node is different from existing server side frameworks because these frameworks use synchronous architecture, where as Node uses an asynchronous architecture. This means that the server reacts and sends messages to the database. The other thing that makes Node different id the use of Javascript as the programming language.
